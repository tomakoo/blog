Hugoの導入手順

Mar 14, 2021
1 min read

Hugoインストール

homebrewのインストール https://brew.sh/index_ja homebrewを用いてHugoをインストール

brew install hugo

※https://gohugo.io/getting-started/quick-start/ クイックスタート参照

サイトを作成

　以下を作業フォルダにて実行 　hugo new site dev-blog

サイトルートディレクトリへ移動 cd dev-blog

テーマ追加

リポジトリをthemes/ディレクトリに複製 git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

次の行をconfig.tomlファイルに追加します。 theme = “m10c”

config.tomlファイルの設定

テーマのインストールの手順に従いテーマを追加する。 https://themes.gohugo.io/hugo-theme-m10c/

ページの作成

hugo new posts/first_post.md

サーバーを立ち上げて確認

hugo server -D
